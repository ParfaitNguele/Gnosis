Comment fonctionne Gnosis ?
Lisez attentivement les règles d'utilisation ci-dessous pour une utilisation efficace du logiciel.

Gnosis, l'outil qui vous permet de rechercher les occurrences d'un thème dans la Bible Essénienne. Suivez ces quelques étapes pour une meilleure utilisation.

Entrer un terme, une expression, une combinaison ou un chiffre pour la recherche

La première étape consiste à saisir votre recherche sur le champ ci-dessous.

Votre recherche peut être :

  1. Un terme simple : Dieu, Amour ou Famille, par exemple.
  2. Une expression : Amour de Dieu, rites esséniens, porteur d'Ange, par exemple.
  3. Une combinaison de deux termes ou expressions : Dieu+Amour ou Rites esséniens+Amour de Dieu, par exemple.
  4. Un chiffre entouré de guillemet : par exemple pour trouver ce qui concerne le chiffre 7 (7 flammes, 7 couleurs, etc.) tapez "7" dans la barre de recherche. Cela ne prendra pas en compte les numéros de versets ni des psaumes. Ce type de recherche est valable pour un seul chiffre (ou nombre) par recherche.

Pensez à saisir la bonne orthographe (accents, singulier, pluriel, etc.) pour avoir de meilleurs résultats.

Notez par ailleurs que le logiciel peut trouver le féminin, le pluriel et les dérivés de la plupart des mots et verbes, et seulement si leur transformation se finit par -s, -e, -es, -ne, -nes, -sse, -sses, -le, -les, -nt, -z, -ssent, -ment, -ments, -lement (petit-es, éternel-les, réel-lement, joue-nt-z, etc.)

Les caractères spéciaux autorisés sont + (pour les recherches combinées), " " (pour les recherches de chiffres), - (pour les mots composés) et ' (pour les groupes de mots comme "Porteur d'Ange").

Filtrer les résultats attendus

Utilisez les cases à cocher pour filtrer les résultats attendus. À ce sujet, vous avez deux catégories de filtres : d'une part, vous pouvez choisir entre "Psaumes" et "Prières" ou alors les deux ensemble (Par défaut, les deux cases sont cochées.) et d'autre part, vous devez choisir au moins un Archange "MICHAËL", "RAPHAËL", "GABRIEL" ou "OURIEL" (Par défaut, les 4 sont choisis.). Lancer la recherche

Pour cela, il suffit de cliquer sur la touche "ENTRER" (sur smartphone comme sur ordinateur) ou sur le bouton "Rechercher".

Exploiter les résultats

Les résultats de votre recherche apparaîtront juste en dessous. Ils sont classés par ordre décroissant (du nombre d'occurrences).

Affichage complet et téléchargement.

Si vous souhaitez parcourir intégralement un contenu, il vous suffit de cliquer dessus. Une fenêtre contextuelle s'ouvre. Vous pouvez ainsi le lire. Votre recherche est surlignée (comme ceci : Amour, Dieu) pour une meilleure visibilité.

Note : les annotations 1,2... apparaissant à la fin de certains mots ("Amour1", "Dieu2") font référence à des notes se trouvant dans la Bible papier. Ces notes n'apparaissent pas dans les textes à lire ou à imprimer via Gnosis.

Si vous souhaitez sauvegarder ou imprimer ledit contenu, cliquez sur le bouton "Télécharger" qui se trouve au-dessous de la fenêtre contextuelle. Votre contenu sera converti en fichier PDF que vous pourrez imprimer une fois le téléchargement effectué.

Pour fermer la fenêtre, cliquez sur "Fermer".
