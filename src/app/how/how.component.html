Comment ça marche ?

Gnosis, l'outil qui vous permet de rechercher les occurrences d'un thème dans la Bible Essénienne. Suivez ces quelques étapes pour une meilleure utilisation.

Entrer un terme, une expression, une combinaison ou un chiffre pour la recherche

La première étape consiste à saisir votre recherche sur le champ ci-dessous.

Votre recherche peut être :

  1. Un terme simple : Dieu, Amour ou Famille, par exemple.
  2. Une expression : Amour de Dieu, rites esséniens, par exemple.
  3. Une combinaison de deux termes ou expressions : Dieu+Amour ou Rites esséniens+Amour de Dieu, par exemple.
  4. Un chiffre entouré de guillemets doubles ; comme ceci : "6". Ici, vous recherchez les occurrences dudit chiffre dans le contenu (numéro de Psaume exclus).

Pensez à saisir la bonne orthographe(accents, singulier, pluriel, etc.) pour avoir de meilleurs résultats.

Notez par ailleurs que : lorsque vous entrez un mot au singulier, Gnosis trouvera les occurrences de ce mot au pluriel. À savoir ceux auxquels on ajoute un "s" ou "x". C'est également le cas pour le féminin (en "elle", "esse", "ne", "e" ; immortelle, prêtresse, bonne, petite).

Les caractères spéciaux autres que + et " sont interdits !

Filtrer les résultats attendus

Utilisez les cases à cocher pour filtrer les résultats attendus. À ce sujet, vous avez deux catégories de filtres. D'une part, vous devez choisir entre "Psaumes" et "Prières" ; ou alors les deux. Par défaut, les deux cases sont cochées. D'autre part, vous devez choisir au moins un Archange (MICHAËL, RAPHAËL, GABRIEL ou OURIEL). Par défaut, les 4 sont choisis. Ci-dessous, un aperçu desdites cases à cocher.

PSAUMES PRIÈRES

Lancer la recherche

Pour cela, il suffit de cliquer sur la touche "ENTRER" (Smartphone comme Ordinateur) ou sur le bouton "Rechercher".

Exploiter les résultats

Les résultats de votre recherche aparaîtront juste en dessous. Ils sont classés par ordre décroissant (du nombre d'occurrences).

Affichage complet et téléchargement.

Si vous souhaitez parcourir intégralement un contenu, il vous suffit de cliquer dessus. Une fenêtre contextuelle s'ouvre. Vous pouvez ainsi le lire.

Si vous souhaitez sauvegarder ou imprimer ledit contenu, cliquez sur le bouton "Télécharger" qui se trouve au-dessous de la fenêtre contextuelle. Votre contenu sera converti en fichier PDF; le téléchargement lancé. Vous pourrez ainsi l'imprimer.

Cliquez sur "Fermer" pour fermer la fenêtre.
